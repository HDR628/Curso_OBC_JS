Typescript 

Linguagem de Programação
JS com tipagem (foi construído em cima)
Fortemente tipada(não consegue modificar as variaveis posteriormente)

Bom para projetos grandes
necessário ter um compilador pra transcrever pra JS(navegador e node)


Tipos primitivos a bit different

boolean ->  let example: boolean = true

number -> let example: number = 10

string -> let example: string = 'Hello'

arrays -> (se for string tu bota string falou)
let example: number[] = [1,2,3]
or 
example: Array<number> = [1,2,3]

Evitando Tipagem


Utilizando o ?  (Seria algo como opcional, tipo "tem ai ou não,senão sem problema" )
function sendspaceship(spaceship{pilot: "String", copilot:? "String"}){
	//codigo
}

Utilizando unknow ( E como falar que não sabe oq vai receber/atribuir a ele, ate mesmo recebendo outras variaveis tipadas, no caso ali tem o text:String)

let input: unknow

input = "Funciona"
input = 1
...

let text: String
text = input (N funciona)
input = text (Funciona)

Utilizando any (E como se o TS parasse de funcionar e deixar todo mundo passar, ñ e recomendado usar)


let input: any

input = "Funciona"
input = 1
...

let text: String
text = input (Agr funciona)
input = text (Funciona)
